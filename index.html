CSS Display & Positioning

The Flow of HTML

The flow of HTML is defaulted as left to right and top to bottom. This is how it exists in the document and that would be considered the "flow" of elements in HTML.

In addition to the properties that it provides to style HTML elements, CSS includes properties that can change how a browser positions elements too. Their properties specify where an element is located on the page, if the element shares lines with other elements, and other related attributes.

Positioning Values

Value Definition
Static This is the default value of an element and doesn't need to be specified.
Relative This value allows you to position an element relative to its default static position on the webpage.
Absolute When an element's position is set to this, all other elements on the page will ignore the element and act like it's not on the page.
Fixed This "fixes" an element to a specific position on the page regardless of scrolling.
Sticky This position value keeps an element in the document flow as the user scrolls but "sticks" to a specified position as the page is scrolled further.
Z-Index The Z-Index is its own property that accepts integer values that control how far forward or how far back an element should appear on the webpage when elements overlap.

How to Write Position Values

  • position: relative;
  • position: absolute;
  • position: fixed;
  • position: sticky;
  • z-index: 1;

Display Values

Value Definition
Inline Inline elements have a box that wraps tightly around their content, only taking up the amount of space nessecary to display their content and not requiring a new line after each element. The height and width of these elements cannot be specified in the CSS document.
Block These elements fill the entire width of the page by default, but their width property can also be set. These elements are not displayed in the same level as the content around them. Unless specified otherwise, they are the height necessary to accomodate their content.
Inline-Block This value combines features of both inline and block elements. Inline-block elements can appear next to eachother adn we can specify their dimensions using the width and height properties.

How to Write Display Values

  • display: inline;
  • display: block;
  • display: inline-block;

Other Layout Properties

Property Definition
Float This simply moves an element as far left or as far left as possible in the container. This is commonly used for wrapping text around an image. (Note: moving elements left or right for layout purposes is better suited for tools like CSS grid and flexbox)
Clear When multiple floated elements have different heights, it can affect their layout on the page. The Clear property specifies how elements should behave when they bump into eachother on the page. Clear can take on one of the following values: left, right, both, or none.

How to Write These Properties

  • float: right;
  • clear: left;


When combined with an understanding of the box model, positioning can create visually appearing web pages.
